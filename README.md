# HireMe 
A personal project designed to aggregate job listings from multiple platforms such as LinkedIn, Instahyre, and Wellfound. The portal provides users with a unified platform to search for jobs, apply directly, and manage their applications efficiently.

---

## Features ‚ú®  
- **Job Aggregation**: Collects job postings from multiple platforms.  
- **Custom Search**: Allows users to search and filter jobs based on preferences (location, role, salary, etc.).  
- **Direct Applications**: Users can apply to jobs directly through the platform.  
- **Application Tracking**: Monitor the status of submitted applications in real-time.  
- **Notifications**: Get notified about new jobs and application updates.
